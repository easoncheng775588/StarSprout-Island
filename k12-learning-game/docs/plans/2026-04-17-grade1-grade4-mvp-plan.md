# 一至四年级 MVP 扩展计划

## 目标

在当前幼小衔接 Learning Island 基础上，连续扩展一年级、二年级、三年级、四年级的课程 MVP，并保持统一的前后端课程结构、互动玩法与家长端联动。

## 总体策略

1. 先补“按学段过滤课程”的底座
2. 再按学段逐个扩内容，不做一次性大爆炸改造
3. 每个学段都遵循同一交付模板：
   - 学科范围定义
   - 章节与关卡种子数据
   - `LevelPlayer` 互动配置
   - 前端展示与导航
   - 自动化测试
   - 验证与提交

## 分阶段范围

### 阶段 A：学段化课程底座

- 孩子 `stageLabel` 支持：幼小衔接、一年级、二年级、三年级、四年级
- 章节数据增加学段字段
- 首页推荐关卡按学段过滤
- 学科地图按学段过滤
- 关卡前进顺序按学段过滤
- 家长端学科总关卡统计按学段过滤

### 阶段 B：一年级 MVP

- 数学：
  - 100 以内数的认识
  - 20 以内进位加法
  - 20 以内退位减法
  - 认识整点/半点
- 语文：
  - 汉字组词
  - 拼音拼读提升
  - 看图读句
  - 标点启蒙
- 英语：
  - 高频词扩展
  - CVC 拼读入门
  - 简单问候句
  - 场景单词配对

### 阶段 C：二年级 MVP

- 数学：表内乘法、长度单位、简单应用题、时间综合
- 语文：词语搭配、句子排序、看图表达、朗读理解
- 英语：更多自然拼读、句型理解、日常对话、绘本理解

### 阶段 D：三年级 MVP

- 数学：除法启蒙、周长面积初步、分数初步、多步应用题
- 语文：段落理解、近反义词、修辞感知、阅读表达
- 英语：句型变换、阅读理解、词组搭配、主题表达

### 阶段 E：四年级 MVP

- 数学：小数初步、角与图形、运算综合、策略应用题
- 语文：篇章理解、写作表达、古诗积累、语法规范
- 英语：时态初步、短文理解、话题表达、综合练习

## 交付节奏

- 本轮先完成：
  - 学段化课程底座
  - 一年级 MVP 第一批内容
- 后续在同一线程继续推进二到四年级
