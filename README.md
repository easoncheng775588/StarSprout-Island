# StarSprout-Island
在游戏里点亮语数英的第一颗星。


当前功能概览

项目现在叫 星芽岛 StarSprout Island，已经从最初的幼小衔接 MVP 扩展到 幼小衔接 + 一年级 + 二年级 + 三年级 + 四年级 的阶段化学习游戏 MVP。前端是 React + Vite + TypeScript，后端是 Spring Boot + JPA，当前本地默认 H2，已预留 MySQL profile。

学生端核心功能

支持登录、注册家长账号、创建孩子档案、切换孩子、退出登录。
孩子档案支持选择学段：幼小衔接、一年级、二年级、三年级、四年级。
首页会根据当前孩子学段推荐下一关，而不是把所有年级内容混在一起。
三大学习岛已建立：数学岛、语文岛、英语岛。
每个学习岛的章节和关卡会按孩子当前学段过滤展示。
完成本关后有奖励反馈、星星奖励、徽章提示和进入下一关的鼓励动画/引导。
每个页面都有返回按钮，不再依赖浏览器后退切换。
课程内容覆盖

幼小衔接已覆盖：

数学：数字认识、11-20、10/20 以内加法、减法、数量比较、规律、图像列式、故事应用题等。
语文：常见字、拼音、声母、笔画、笔顺、简单识字。
英语：26 个字母分组、自然拼读、生活单词、颜色单词、绘本跟读。
一年级 MVP 已覆盖：

数学：100 以内数、20 以内进位加法、退位减法、生活应用题。
语文：生字组词、拼音拼读提升、看图读句、标点启蒙。
英语：校园单词、CVC 拼读、问候句、动作单词。
二年级 MVP 已覆盖：

数学：表内乘法、长度单位、两步应用题、时间推算。
语文：词语搭配、句子排序、看图表达、朗读理解。
英语：自然拼读进阶、句型理解、日常对话、绘本理解。
三年级 MVP 已覆盖：

数学：除法平均分、周长、分数初步、多步应用题。
语文：段落理解、近反义词、修辞感知、阅读表达。
英语：句型变换、词组搭配、阅读理解、主题表达。
四年级 MVP 已覆盖：

数学：小数初步、角与图形、运算综合、策略应用题。
语文：篇章理解、写作表达、古诗积累、语法规范。
英语：时态初步、短文理解、话题表达、综合练习。
互动玩法

目前已实现多种可交互玩法：

数字/答案选择
拖拽或点击式数量计数
拿走物品做减法
图形/颜色规律选择
数量比较
图像列式
故事应用题
汉字/词语选择
拼音和英语听辨
字母/单词/句子跟读
单词图片配对
笔顺点击
绘本/短文朗读
音频目前使用浏览器语音合成，支持中英文朗读和慢速跟读模式，但素材音质还属于 MVP 级别，后续可以换成真人或高质量 TTS 音频。

家长端功能

家长中心可以查看孩子学习概览。
支持今日完成关卡、学习时长、获得星星。
支持学科进度、周学习趋势、薄弱点建议。
支持学习活跃度、平均正确率、平均学习时长、最佳学习时段等数据维度。
支持按当前孩子学段统计课程总量和完成情况。
支持家长设置：排行榜参与开关、每日学习目标、提醒开关。
成就和排行榜

有成就系统，包含已解锁和进行中的徽章。
完成关卡会结算星星和徽章。
重复练习不会重复刷有效星星。
排行榜有真实结算逻辑，支持：
本周星星榜
连续学习榜
挑战达人榜
家长可以关闭排行榜参与，关闭后孩子不参与排名展示。
数据与持久化

后端已从前期 mock 逐步迁移到 JPA Repository + 数据库种子数据。
本地测试使用 H2。
已配置 MySQL profile，后续可以接正式 MySQL。
账号、孩子档案、当前孩子选择、学习完成记录、家长设置等都已走后端持久化链路。
